The application is a catalog listing for movies & TV 
You can register for an API key https://developers.th getting-started/introduction

You are to follow the detailed instructions as below

installation for android

open terminal
- cd testTmdbmovie
- npm install 
- react-native start
- react-native run-android

installation for ios

open terminal
- cd testTmdbmovie
- cd ios
- pod init 
- pod install 
- react-native start
- react-native run-ios //--simulator="iPhone version"
